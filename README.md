## virtual environment
To create virtual environment in `v:\mlvenv` folder

```
$ py -m venv v:\mlvenv
```

## packages
To install all packages listed in `requirements.txt`

```
$ v:\mlvenv\Scripts\activate
$ pip install -r requirements.txt
```

## jupyter notebook
To launch Jupypter Notebook

```
$ jupyter notebook
```
